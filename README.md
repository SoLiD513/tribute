## Khabib Tribute

A page built in honor of the UFC Lightweight Champion KHABIB NURMAGOMEDOV "The Eagle"

The design and layout are taken from the offial UFC page. The gallery contains fan-made images. 

Fan comments can be posted in the comments section &
the database is powered by Google's Firestore. 

## Getting started

In the project directory, you can run:

### `yarn install`
### `yarn start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.<br>
You will also see any lint errors in the console.
